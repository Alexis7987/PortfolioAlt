Procédures Techniques

Procédures réalisées pendant le BTS SIO

Installation de VS Code

  1. Télécharger l'installateur sur le site officiel
  2. Lancer l'installation sur Windows
  3. Mettre l'interface en français et choisir un thème
  4. Ajouter des extensions (Live Server, Prettier...)

Configuration Git & GitHub

  1. Installer Git Bash sur Windows
  2. Taper les commandes pour configurer son pseudo et mail
  3. Créer un repo sur le site GitHub
  4. Utiliser git add, commit et push pour envoyer le code

Base de données MySQL

  1. Démarrer UwAmp sur le PC
  2. Ouvrir l'interface de phpMyAdmin dans le navigateur
  3. Créer la base de données et les tables
  4. Faire quelques requêtes de test en SQL

Configuration BeyondTrust

  1. Lancement de l'outil sur le poste
  2. Prise en main à distance des PC des utilisateurs
  3. Vérification des accès et droits
  4. Respect des bonnes pratiques de sécurité

Déploiement Whisper (OpenAI)

  1. Installer Python et vérifier la variable PATH
  2. Taper pip install openai-whisper
  3. Télécharger et faire marcher le modèle "base"
  4. Écrire un petit script Python pour utiliser l'outil

Projet Symfony

  1. Installer Composer sur Windows
  2. Taper la commande symfony new pour créer le projet
  3. Connecter le projet à la base de données dans le .env
  4. Créer les contrôleurs et les pages Twig simples
