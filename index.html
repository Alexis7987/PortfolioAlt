Résumé de ma formation en Services Informatiques aux Organisations

BTS Services Informatiques aux Organisations

Institution Saint-Aspais — Melun (77)

Le BTS SIO (Services Informatiques aux Organisations) est un diplôme reconnu par l'État de niveau Bac+2. Cette formation prépare les étudiants aux métiers de l'informatique dans les organisations, avec une approche à la fois technique et professionnelle.

La formation se déroule sur 2 ans et inclut des périodes de stage en entreprise permettant de mettre en pratique les compétences acquises.

Objectifs de la formation

  • Participer à la production et à la fourniture de services informatiques
  • Installer, intégrer et administrer des solutions informatiques
  • Gérer le patrimoine informatique d'une organisation
  • Répondre aux incidents et aux demandes d'assistance
  • Développer la présence en ligne de l'organisation
  • Mettre à disposition un service informatique sécurisé

Procédures Techniques

Procédures informatiques réalisées en BTS SIO

01

Installation de Visual Studio Code

  1. Téléchargement depuis le site officiel de Microsoft
  2. Installation pas à pas sous Windows
  3. Configuration de base et personnalisation
  4. Installation des extensions essentielles (Live Server, PHP IntelliSense, Python)
02

Configuration GitHub & Git

  1. Installation de Git sur Windows
  2. Configuration initiale (nom, email, clé SSH)
  3. Création d'un dépôt et premier commit
  4. Commandes de base : clone, pull, push, branch
03

Mise en place d'une base MySQL

  1. Installation de UwAmp / WAMP
  2. Accès à PhpMyAdmin
  3. Création d'une base de données et de tables
  4. Écriture de requêtes SQL (SELECT, INSERT, JOIN)
04

Configuration BeyondTrust

  1. Présentation de l'outil d'accès sécurisé
  2. Configuration des accès distants aux postes
  3. Gestion des droits et des permissions
  4. Bonnes pratiques de cybersécurité
05

Déploiement de Whisper (OpenAI)

  1. Installation de Python et des dépendances
  2. Installation locale de Whisper via pip
  3. Configuration du modèle de transcription
  4. Automatisation via un script Python
06

Création d'un projet Symfony

  1. Installation de Composer et Symfony CLI
  2. Création d'un nouveau projet Symfony
  3. Configuration de la base de données (Doctrine)
  4. Création d'entités, controllers et vues Twig
